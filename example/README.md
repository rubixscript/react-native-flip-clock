# Flip Clock Example

A simple React Native Expo app to test the `@rubixscript/react-native-flip-clock` component.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Run on your preferred platform:
   - iOS: `npm run ios`
   - Android: `npm run android`
   - Web: `npm run web`

## Usage

This example demonstrates a simple usage of the FlipClock component with:
- 25-minute Pomodoro timer
- Dark theme
- Control buttons
- Manual start option
- Time completion callback
